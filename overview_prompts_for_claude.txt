IlluMefy アプリ開発コンセプト概要
基本コンセプト
IlluMefyは、フォークソノミー（ユーザー主導のタグ付け）を活用してゲームインフルエンサーを発見・分類するモバイルアプリ。「Illuminate（照らす）」+「Me（自分）」+「-fy（他者に広げる）」の造語で、ユーザー自身と他者の好みを照らし出すプラットフォーム。
核となる価値

情報の民主化と集合知の形成 - ユーザーが自由にタグ付けを行い、インフルエンサーの多様な特性や価値を浮き彫りにする
タグを通じた新しい発見体験 - 主観的な評価を集合知として蓄積し、新たな発見経路を提供
コミュニティ形成と価値観の共有 - 似たタグ付けをするユーザー同士が繋がり、新たなコミュニティを形成

フェーズ1の主要機能

インフルエンサー基本情報 - プロフィール、統計情報の表示
フォークソノミー核機能 - ユーザーによるタグ追加、タグクラウド表示、タグベース検索、タグへの共感/評価
検索・発見機能 - インフルエンサー名検索、タグによる検索、人気度順表示
ユーザー管理 - 登録・ログイン、簡易プロフィール設定、タグ付け履歴
初期データセット - 人気ゲームインフルエンサーの基本情報、初期タグ
モデレーション基盤 - 不適切タグ報告、簡易自動フィルタリング、管理者によるタグ審査
シンプルなレコメンデーション - 類似タグを持つインフルエンサー表示

主要画面

ホーム/ダッシュボード画面 - Netflix風のレコメンデーション型UI、人気タグクラウド、おすすめインフルエンサー
検索・探索画面 - インフルエンサー検索、タグベース検索
インフルエンサープロフィール画面 - 基本情報、タグクラウド、タグ追加機能
ユーザー認証・マイページ画面 - ログイン、タグ付け履歴
タグ管理・操作画面 - タグ追加、評価、報告

データモデル概要

Influencer: id, name, platforms, statistics, tags, popularity
Tag: id, name, count, createdBy
Tagging: influencerId, tagId, userId, agreements
User: id, username, createdTags, appliedTaggings

技術スタック案

フロントエンド: React Native
バックエンド: Node.js + Express
データベース: MongoDB（柔軟なスキーマ）
認証: Firebase Authentication
API: RESTful API

開発フェーズ
フェーズ1（基本機能）: 1年以内に実装可能

個人開発: 7-11ヶ月
小規模チーム: 5-7ヶ月

ユーザー体験の特徴

Netflix風のレコメンデーションUI
ユーザー主導でタグを作成・評価
集合知によるインフルエンサー発見
相対評価によるタグの質向上

開発戦略

MVPを早期にリリースしてフィードバックを得る
初期データとして100-200人のインフルエンサー情報を手動収集
ユーザーによるタグ付けを促進する設計
段階的な機能拡張
